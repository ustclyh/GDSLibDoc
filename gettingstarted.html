Get Started

程序包包含 ElementStructures, Devices以及ChipConstructors 三个层级.

ElementStructures 封装了 gdstk 的三种对象, 即 gdstk.RobustPath, gdstk.Polygons 和 gdstk.Reference. ElementStructures中的类 Paths, Polygons 和 References 均包含了存储对应 gdstk 对象的列表. 此外 Paths 可单独使用, 用于生成简单结构, 方便基于 FDTD 的仿真.

Devices 提供了构造器件的方法, 它是 gdstk.RobustPath, gdstk.Polygons 和 gdstk.Reference 的子类, 继承了一切属性和方法. Devices 提供的方法 generate_cell 将自动融合其三个父类提供的列表, 将全部对象放置于一个 PCell 内, 并将此 PCell 注册于一可以随时调用的字典, 方便后续的芯片级操作.

ChipConstructors 提供了多种生成 Devices 引用的方法, 完成芯片的最终 gds 导出.

下面将按照上述层级对程序包的使用进行介绍.

ElementStructures 层级

Paths 类

Waveguide 类调用 gdstk.RobustPath 生成波导几何结构. 我们将结合例子说明此类的基本使用方法.

import numpy as np
import gdstk
from ElementStructures import Paths

lib = gdstk.Library('library')

path = Paths()

# plot a racetrack ring
path.new_path(start=[0, 5], width=2, layers=0)
path.add_segment(length=30, direction=0)
path.add_turn(radius=10, angle=np.pi)
path.add_segment(length=30)
path.add_turn(radius=10, angle=np.pi)
# add a copy of current path, otherwise the follow mirror operation will influence this added path
# change the 'add_copy' parameter to 'false' to see the different
path.end_current_path(add_copy=True)

# mirror the above ring and add it to the path list
path.curPath.mirror([1, 0])
path.end_current_path()

# plot another waveguide, which is a straight one
path.new_path(start=[-35, 0], width=1, layers=1)
path.add_segment(length=100, direction=0)
path.end_current_path()

# generate PCell and add it to the GDS library
lib.add(path.generate_pcell('MAIN'))
# write GDS file
lib.write_gds('example.gds')

path = Paths() 用于类的初始化, 随后我们便可以反复调用 Paths 提供的方法生成结构. 每当需要生成一条新路径, 调用 new_path 方法开始路径, 随后便可反复调用 add_* 方法加入不同结构, 当完成一条路径绘制后, 调用 end_current_path 方法结束绘制. 可以通过以上过程多次添加不同路径. 每一对 new_path 和 end_current_path 都会为 Paths.path_list 添加一个新的元素. 当完成全部路径绘制后, 调用 generate_pcell 方法, 便可将列表中已添加的全部路径对象放置在一个 gdstk.Cell 中, 此方法将放置路径后的 gdstk.Cell 对象作为返回值返回.

Note

对于熟悉 gdstk 包的用户, Paths 还可以以一种更灵活的方式使用. end_current_path 方法的参数 add_copy 若为真, 则加入路径列表的元素为当前操作路径对象的拷贝. 此时可以直接操纵 Paths 类的属性 curPath, 使用 gdstk.RobustPath 提供的方法, 对其进行镜像, 旋转等操作, 随后再次调用 end_current_path 便能够将变换后的路径再次加入 Paths.path_list.

Polygons 和 References 类

这两个类用于构建 Devices 类, 不建议用户单独调用, 不提供操作说明. 如果有需要, 可以自行阅读代码 (代码量很小).

Devices 层级

Device 类

Device 为定义任意器件提供了通用的方法和参数.

基于 Device 类的器件构造方法

from Devices import Device
import gdstk
import numpy as np


# Parent class of all the devices object.
class CircleExample(Device):

    def __init__(self, name):
        super().__init__(name)
        self._position = [0, 0]
        self._radius = 5
        self._layer = 1

    ##############################################################################
    # All the parameters for defining a pattern should be realized with decorators
    # Only in this way, the dependency of parameters can be resolved
    @property
    def position(self):
        return self._position

    @property
    def radius(self):
        return self._radius

    @property
    def layer(self):
        return self._layer

    @position.setter
    def position(self, value):
        self._position = value
        self.clearall()

    @radius.setter
    def radius(self, value):
        self._radius = value
        self.clearall()

    @layer.setter
    def layer(self, value):
        self._layer = value
        self.clearall()

    # End of parameter definition
    ####################################################################################

    # draw to generate geometry (PolygonSet)
    def draw(self):
        rnd = gdstk.ellipse(self._position, self._radius/3, initial_angle=0, final_angle=2*np.pi,
                            layer=self._layer,
                            tolerance=1e-4)
        self.add_polygon(rnd)
        rnd = gdstk.ellipse(self._position + np.array([10, 10]), self._radius, initial_angle=0,
                            final_angle=2 * np.pi,
                            layer=self._layer,
                            tolerance=1e-4)
        self.add_polygon(rnd)

if __name__ == "__main__":
    # Example
    # initialize the library
    lib = gdstk.Library()
    # generate segment path
    testCircle = CircleExample("TEST")
    testCircle.draw()
    testCircle.draw_box()
    # generate the PCell and put the PCell in library
    lib.add(testCircle.generate_pcell('FIRST'))
    # write the gds file
    lib.write_gds("first.gds")
