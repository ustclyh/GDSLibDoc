Installation

Python 环境

推荐使用 Anaconda3 搭建 Python 环境，Anaconda3 作为软件直接安装即可。

IDE

推荐使用 PyCharm Community，直接官网下载并安装即可。

配置

Anaconda3 与 PyCharm Community 均安装完毕后，打开 PyCharm Community ，新建工程，工程名即文件夹名 （例如 GDSLib_gdstk，下文将始终使用该名称示例）。 展开下方 Python Interpreter 选项卡，选择 New environment using conda ，Python 版本选择最新 3.10。 点击最下方创建。

该过程将自动为我们在 conda 中创建名为 GDSLib_gdstk 环境。 Anaconda 将 Python 环境沙盒化，能够同时在一台电脑中创建多个独立的 Python 环境，多个环境可以使用 不同的 Python 及模块版本而不产生冲突，十分有利于多个基于不同 Python 版本的项目的同时开发。

在开始菜单中，进入 Anaconda Powershell Prompt (Anaconda3)。 分别输入以下命令，第一行激活 PyCharm 为我们创建的环境，随后安装可能需要的模块

conda activate GDSLib_gdstk
conda install gdstk
conda install numpy
conda install scipy
conda install matplotlib

至此，Python环境安装完成。
